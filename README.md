# 🎬 CineLog - CP3: CRUD e IA Generativa

Este repositório contém a entrega do **CP3** da disciplina de Web Development (FIAP). A proposta da atividade foi criar uma aplicação web que simula um sistema de cadastro de itens (CRUD), utilizando o auxílio de Inteligências Artificiais Generativas e seguindo a metodologia de *Spec-Driven Development*.

O tema escolhido para este projeto foi um gerenciador simples de **Lista de Filmes Assistidos**.

## 🚀 Funcionalidades

A aplicação possui uma interface limpa e as seguintes funcionalidades implementadas em JavaScript puro:

* **Login:** Tela de acesso inicial com validação de campos vazios e credenciais corretas.
* **Create:** Adição de novos filmes tanto no *início* quanto no *final* da lista.
* **Read:** Renderização dinâmica da lista usando manipulação de DOM (`innerHTML` com `.map()`).
* **Update:** Edição individual do nome de cada filme salvo.
* **Delete:** Remoção de filmes específicos baseada no índice do array.

## 🛠️ Tecnologias Utilizadas

Para manter o foco nos fundamentos e cumprir as regras da avaliação, não foram utilizados frameworks ou bibliotecas externas.

* **HTML5:** Estruturação semântica.
* **CSS3:** Estilização simples e responsiva.
* **JavaScript (ES6+):** Lógica da aplicação, manipulação de arrays (`push`, `unshift`, `splice`) e eventos de DOM.

## 🤖 Uso de Inteligência Artificial

A construção do código-fonte contou com o auxílio de IAs Generativas. Todo o processo de criação do *prompt*, análise das respostas do Claude, ChatGPT e Gemini, bem como a justificativa das escolhas técnicas, estão documentados no arquivo de requisitos.
